# 🚀 Quick Start Guide

## Get Started in 3 Steps!

### Step 1: Install Dependencies
```bash
yarn install
```
⏱️ Takes ~2-3 minutes

### Step 2: Start Development Server
```bash
yarn start
```
🌐 Opens at `http://localhost:3000`

### Step 3: Customize Your Data
Open `src/data/mock.js` and update with your information!

---

## 📝 What to Update First

### 1. Personal Information (Line 3-12)
```javascript
export const personalInfo = {
  name: "Your Name",
  email: "your.email@gmail.com",
  linkedin: "https://linkedin.com/in/yourprofile"
}
```

### 2. Skills (Line 20-80)
Update your programming languages, web technologies, AI skills, and tools.

### 3. Projects (Line 82-150)
Replace with your actual projects, descriptions, and technologies used.

### 4. Experience (Line 152-180)
Update with your work experience.

### 5. Education (Line 14-18)
Update your degree and institution.

## 🚢 Deploy Your Portfolio

### Deploy to Vercel (Recommended)
```bash
yarn build
# Then upload 'build' folder to Vercel
```

### Deploy to Netlify
```bash
yarn build
# Drag and drop 'build' folder to Netlify
```
